# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can find the
full documentation for it [in the main changesets repo](https://github.com/changesets/changesets).

**What is a changeset?**

A changeset is a piece of information about changes made in a branch or commit. It holds three bits of
information:

- What we need to release
- What semver bump type to apply (major / minor / patch)
- A summary of the changes made

Learn more about [creating a changeset](https://github.com/changesets/changesets/blob/main/docs/adding-a-changeset.md).
